## Installation
Clone this repo: 
```
git clone github_path_to/GlyphOnly
cd GlyphOnly
```

Build up a new environment and install packages as follows:
```
conda create -n glyphonly python=3.8
conda activate glyphonly
pip install -r requirements.txt
```
